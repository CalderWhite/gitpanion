# gitpanion
![PyPi](https://img.shields.io/badge/pypi-v1.0.0-blue.svg)    
A module to make using the github v3 REST api in python 3 easy.

## Install
### Windows
`py -m pip install gitpanion`
### Linux
`sudo pip install gitpanion`
## Usage
```Python
from gitpanion import github_api
```
